
# Algorithm Visualizer

Algorithm Visualizer is an interactive way and platform that visualize the algorithms in three main domain i.e. Basic Graph Algorithm, Path Finding and Sorting algorithm. The project focuses on visualizing ðŸ“Š the algorithm and making easier to understand and learn the algorithm.

## Authors

- [nightwing808](https://github.com/nightwing808)

## Built with
 - Language : HTML,CSS,JavaScript
 - IDE : VSCode

## Features

#### Sorting Algorithm
 - Insertion Sort
 - Bubble Sort
 - Merge Sort
 - Selection Sort
 - Quick Sort
 - Tim Sort
#### Graph
 - DFS
 - BFS
 - Dijkstra
 - Topological Sort
#### Pathfind (Maze)
 - DFS
 - BFS
 - Dijkstra

## Installation

 - In VSCode, add these extension
```bash
Live Server
JavaScript (ES6) Code Snippets
```

- Install [node.js](https://nodejs.org/en)
- Open Terminal and copy these commands
```bash
cd path/to/your/project
```
```bash
npm install vis-network
```
